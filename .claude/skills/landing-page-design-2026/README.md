# Landing Page Design 2026 Skill

Modern 2026 landing page design patterns following the **Evolution of Todo** hackathon project style.

## Installation
```bash
cp -r landing-page-design-2026 .claude/skills/
```

## Usage
"Create a landing page with hero typography and kinetic motion."
"Setup a dark mode design system for 2026."
"Show me examples of scrollytelling components."

## Features
- **Kinetic Typography**: Large fluid-font headers.
- **Eco-conscious Dark Themes**: Energy-saving OLED-optimized palettes.
- **Anticipatory UX**: Dynamic personalization strategy.
- **Micro-interactions**: Framer Motion implementation patterns.
